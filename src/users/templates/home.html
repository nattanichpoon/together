Together

Together is a web application designed to help you and your classmates manage and distribute the tasks evenly. This web application is tailored specifically for small-scaled collaborative projects.

Get Started

Our Features

Distributed Task Assignment
Tasks for each projects are weighted by difficulty level, and assigned so that all members have equal assignments.
Synchronized Calendars
View shared calendars for each project to aid with finding suitable meeting slots that is convenient for everyone.
Productivity Report and Progress Graphs
Generates a summary of contribution levels, task distribution among members, and project progress. Links can be shared via email.
Peer Ratings
Anonymously rate your peers to provide them with feedback. These feedbacks will be summed up and summarized to each member.
